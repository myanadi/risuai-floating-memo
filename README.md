# 🗒️ Floating Memo — RisuAI Plugin

A floating notepad overlay plugin for [RisuAI](https://github.com/kwaroran/Risuai). 
Preserves user context (settings, character notes, scratch) without leaving the chat screen.

## ✨ Features

- **Folder system** — Organize notes into custom folders
- **Live search** — Filter notes by keyword across all folders
- **Floating overlay UI** — Stays on top of chat without disrupting flow
- **3-tier clipboard fallback** — Works across modern + legacy browsers
- **Mobile-friendly** — Touch-optimized for narrow screens
- **Persistent storage** — Notes saved to local storage (`floating_memo_data_v1`)

## 🚀 Installation

1. Download `floatingmemo.js`
2. In RisuAI: **Settings → Plugins → Import Plugin**
3. Select the downloaded `.js` file
4. Enable the plugin

## 🛠️ Tech Stack

- Vanilla JavaScript (no frameworks)
- RisuAI Plugin API v3.0
- Developed with AI-assisted workflow (Claude/GPT)

## 📝 Why This Plugin

Originally built to solve a personal friction: RP sessions often require referencing 
external notes (character settings, lore, dialogue drafts), but switching apps breaks 
immersion. This plugin keeps a persistent, searchable notepad as an overlay layer 
directly on top of the chat UI.
